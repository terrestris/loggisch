## loggisch

Tiny JS logging library for NodeJS and the browser (ES6/UMD).

## Installation & usage

```sh
bun add loggisch
```
or

```sh
npm i loggisch
```

In your code:
```ts
import log from 'loggisch';
```

## API

Check out the main.ts

## Development

For development bun is recommended:

### Tests (requires bun):
```sh
bun test
```

### Building
```sh
bun run build
```
or
```sh
npm run build
```
